# Tarea2.1_Dise-o-Digital

## API REST de Productos

Este proyecto consiste en una **API REST** desarrollada con **Node.js** y **Express**, que permite realizar operaciones básicas (CRUD) sobre una lista de productos almacenados en un archivo JSON local.

---

## ⚙️ Instrucciones para instalar las dependencias

1. Asegúrese de tener **Node.js** instalado en su computadora.
2. Abra la carpeta del proyecto en **Visual Studio Code**.
3. Ejecute el siguiente comando en la terminal para instalar las dependencias necesarias:

   npm install
   COMO EJECUTAR LA API
   npm run dev
